Learning Objectives
======

* Die Teilnehmer können eine kleinere wissenschaftliche Arbeit unter Anleitung erstellen.
* Sie können Literaturrecherchen zu einem vorgegebenen Thema durchführen
* Sie können die für ein vorgegebenes Seminararbeitsthema relevanten Informationen analysieren, auswählen, mit eigenen Gedanken aufbereiten und daraus eine schriftlichen Arbeit verfassen.
* Sie können ein von ihnen erarbeitetes Thema in Form eines Seminarvortrages vermitteln.
* Sie können Für und Wider bestimmter Techniken, Konzepte, Technologien oder Ideen abwägen, sich ein eigenes Urteil bilden und dieses begründen.
